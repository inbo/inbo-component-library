Autocomplete Component

The INBO Autocomplete component provides an autocomplete input field with search functionality.

Example

Selected country: {{ selectedCountry.name }} ({{ selectedCountry.code }})

Example with Input Mask

The example below demonstrates an input mask with pattern: 2 chars - 2 chars - 2 chars

Selected product: {{ selectedProduct.code }} - {{ selectedProduct.name }}

Usage

<inbo-autocomplete
  label="Label"
  placeholder="Placeholder"
  [searchFunction]="searchFunction"
  displayPattern="{{ '{property1}' }} ({{ '{property2}' }})"
  [showClearIcon]="true"
  (onOptionSelected)="onItemSelected($event)">
</inbo-autocomplete>

With Input Mask

<inbo-autocomplete
  label="Label"
  placeholder="Placeholder"
  [searchFunction]="searchFunction"
  mask="cc-cc-cc"  
  displayPattern="{{ '{property1}' }} ({{ '{property2}' }})"
  [showClearIcon]="true"
  (onOptionSelected)="onItemSelected($event)">
</inbo-autocomplete>
